\sctn{Extras}
\ssctn{Memory Visualization}
\fig{\centering\img{0.5}{memvis}}{Memory visualisation}{memvis}
Found at: \itl{Special \rarr memvis}\\\\
Memory visualization shows the blocks of memory which are occupied. It can also be used to draw pictures.
\pagebreak
\ssctn{Terminal}
\fig{\centering\img{1}{terminal}}{Terminal window}{terminal}
Preconditions: \itl{Running in emulator mode}\\
Found at: \itl{Special \rarr Features}\\\\
When using KoMoDo in emulator mode, a terminal is available which prints the outputs from \func{SWI 3} commands.
\pagebreak

\ssctn{Symbol Window}
\fig{\centering\img{1}{symbol}}{Symbol window}{Symbol}
Found at: \itl{Special \rarr Symbol window}\\
The symbol window displays the symbols or defines for a particular program.
